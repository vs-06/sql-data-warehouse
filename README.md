# sql-Data-warehouse
Building a Medallion Architecture type Data Warehouse with MySQL, including ETL processes, Data Modeling, and Analytics.
The purpose of this project is to implement Data Warehousing using MYSQL database server.

Tasks performed as part of the data warehousing:
1. ELT Processing
2. Data Integration
3. Data Cleansing
4. Data Loading
5. Data Modeling

Extraction Method: PULL Extraction
Extraction Type: FULL Extraction
Extraction Technique: File Parsing
